# Assets

Place these files here before building:

- `icon.png` — 1024x1024 app icon (dark background #0F0F0F with accent-colored icon)
- `splash.png` — 1284x2778 splash screen
- `adaptive-icon.png` — 1024x1024 Android adaptive icon foreground
- `favicon.png` — 48x48 web favicon

You can generate these at https://www.appicon.co/ or use any image editor.
Recommended: a simple green ✓ checkmark on the dark #0F0F0F background.
